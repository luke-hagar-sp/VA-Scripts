# VA-Scripts

## STUNT.sh 
The STUNT script (Support Team Unified Network Tester) assists in troubleshooting IdentityNow VA (Virtual Appliance) problems by gathering information in a snapshot style, and placing it into one file. The stuntlog.txt file will be placed in your home directory (/home/sailpoint/). This software is not officially supported by SailPoint, and comes with no warranty.

## Stunt script: Single Line Deploy
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/luke-hagar-sp/VA-Scripts/main/stunt.sh)"
```
